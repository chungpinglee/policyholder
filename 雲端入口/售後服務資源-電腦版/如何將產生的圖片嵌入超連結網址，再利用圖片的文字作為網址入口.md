#### **使用工具精準框選 (解決文字無法點選)**

因為我無法「看見」您最終圖片上每一個字的精確像素位置，最強烈建議您使用線上的**影像地圖產生器**重新框選一次，這樣準確度就是 100%。

**推薦工具：** [Image Map Generator](https://www.image-map.net/)

1. 上傳您的 `map.png`。
2. 用滑鼠把每一個「文字」和「圖示」框起來。
3. 輸入對應的網址。
4. 點擊「Show Me The Code」取得座標 (`coords="..."`)，貼回下方的 HTML 中。

**如何操作 [Image Map Generator](https://www.image-map.net/)**

### **步驟 1：上傳圖片**

1. 進入該網站（Image Map Generator）。
2. 點擊畫面中間綠色的按鈕 **「Select Image from My PC」**。
3. 選擇您電腦中那張 **`map.png`** (或 `map.jpg`) 圖片上傳。

### **步驟 2：開始框選連結區域**

這是最關鍵的一步，您可以用滑鼠精準畫出您想讓使用者點擊的範圍。

1. 在圖片預覽區上方，會有形狀工具：
   - **Rect (矩形)**：最適合框選方形的按鈕或文字（如「南山人壽官方網站」的招牌）。
   - **Poly (多邊形)**：適合不規則形狀（如斜斜的道路或建築物）。
   - **Circle (圓形)**：適合圓形圖示。
2. **動手畫**：選好形狀後，在圖片上把您想要有超連結功能的「文字」或「圖示」框起來。
   - *小技巧：框框可以畫大一點，蓋過文字和旁邊的圖示，這樣手機上比較好點。*

### **步驟 3：輸入網址 (Link)**

每畫好一個框框，系統會跳出設定欄位：

- **Link (網址)**：請貼上該按鈕對應的網站連結（例如南山人壽官網網址）。
- **Title (標題)**：輸入滑鼠移過去時要顯示的文字（例如「前往官網」）。
- **Target**：建議輸入 `_blank`（代表在新視窗開啟）。

**請重複步驟 2 和 3，把地圖上所有您想設定連結的地方都畫一遍。**

### **步驟 4：取得程式碼**

全部畫完後：

1. 點擊網頁下方的紅色或亮色按鈕 **「Show Me The Code」**。

2. 您會看到一串 HTML 程式碼，看起來像這樣：

   HTML

   ```
   <map name="image-map">
       <area target="_blank" alt="官網" title="官網" href="https://..." coords="34,44,270,350" shape="rect">
       ...
   </map>
   ```

3. 請把這整段 `<map ...> ... </map>` 的內容**複製**下來。

### **步驟 5：替換回原本的 HTML 檔案**

1. 用記事本開啟您原本的 HTML 檔案。
2. 找到原本的 `<map name="service-map"> ... </map>` 這一段。
3. **刪除**原本舊的 `<area>` 內容，貼上您剛剛從網站上複製的新代碼。
4. **重要修正**：新產生的代碼中，`map name` 預設可能是 `image-map`，請記得改回與圖片對應的名字（如果您沒動原本的 img 標籤，請確保 `<img ... usemap="#service-map">` 中的名字與 `<map name="service-map">` 一致）。
